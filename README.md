# Productivity Prediction of Garment Employees

## Overview

This project focuses on building a classification model for predicting garment employee productivity. The goal is to develop an accurate and reliable model to assess employee performance based on various factors.

## Project Objectives

* Pre-process and clean the garment employee productivity dataset for optimal model performance.
* Develop and compare classification models using various machine learning algorithms.
* Evaluate model performance using key metrics such as accuracy, precision, recall, and F1-score.
* Optimize model performance through hyperparameter tuning.

## Key Features

* **Data Preprocessing:** Cleaned and prepared the dataset to ensure data integrity and quality.
* **Machine Learning Models:** Implemented Decision Tree and Random Forest for classification.
* **Performance Evaluation:** Compared models based on accuracy, precision, recall, and F1-score.
* **Comparative Analysis:** Identified the best-performing model for reliable productivity predictions.

## Tools & Technologies Used

* **Python** for data analysis and modeling
* **Pandas & NumPy** for data preprocessing
* **Scikit-learn** for implementing machine learning models
* **Matplotlib & Seaborn** for data visualization

## Results & Insights

* Identified the best-performing model based on evaluation metrics.
* Improved prediction accuracy through data preprocessing and hyperparameter tuning.
* Provided valuable insights into garment employee productivity through feature analysis.
